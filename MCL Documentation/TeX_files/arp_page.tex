\chapter{Arpeggiator Page:}
The Arpeggiator Page is allows control of the Arpeggiator function located in the Chromatic page.
\screenshot{chro_menu.png}


\textit{To enter the Arpeggiator Page: From the Chromatic Page, enter the Track Menu by holding \textbf{[Shift 2]}. Select Arpeggiator from the Track Menu }

\screenshot{arp_page.png}

The Arpeggiator has 3 states of operation:
\begin{enumerate}
    \item -- (Off)
    \item ON
    \item LATCH (Notes are held down)
\end{enumerate}

When enabled, the arpeggiator will  play when the sequencer is running.
